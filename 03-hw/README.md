# node-tasks

## Final, 3rd task

Uber-like web application for freight trucks using REST, Vue.js and MongoDB.

## Prerequisites

Installed Node.js, access to MongoDB database.

## Project setup

### Client (UI side)

```
cd client
npm i
npm run serve
```

### Server (BE side)

```
cd server
npm i
npm run start
```
